# Leck Electrical — Add Your Chat Assistant to Your Website

Your site: **WordPress + Divi theme**. Takes about 2 minutes. No plugin needed.

---

## Step 1 — copy this code

```html
<script src="https://chat.barnettdigital.co/chat-widget.js?v=5"></script>
<chat-widget
  url="https://chat.barnettdigital.co/chat"
  agent="Leck Electrical"
  business="Auckland electricians"
  avatar="https://chat.barnettdigital.co/assets/leck-logo.png"
  tagline="Typically replies instantly"
  greeting="Hey 👋 Leck Electrical here — Auckland electricians. Got a sparky job or after a quote? Ask me anything, or I'll get Bryce to call you back."
  prompts="Get a quote|Do you do EV chargers?|Switchboard upgrade?|Do you cover my area?"
  accent="#d8a963"
  header="#1a1a1a"
  header2="#d8a963"
  teaser="👋 Send us a message — we're online!"
  teaser-timeout="5000"
></chat-widget>
```

## Step 2 — paste it into your site (Divi — easiest way)

1. Log in to **leckelectrical.co.nz/wp-admin**
2. In the left menu, go to **Divi → Theme Options**
3. Click the **Integration** tab along the top
4. Paste the code into the box labelled **"Add code to the < body >"**
5. Click **Save Changes**

*(Prefer a plugin instead? Install the free **"WPCode"** plugin → **Code Snippets →
Header & Footer** → paste into the **Footer** box → **Save**. Either way works.)*

## Step 3 — check it worked

Open **leckelectrical.co.nz** in a **private/incognito window** and wait ~2 seconds —
a gold chat bubble appears in the bottom-right, with a little "Send us a message" pop-up.
Click it and send a test message. Done. ✅

If you don't see it: hard-refresh with **Ctrl/Cmd + Shift + R**, and double-check you
clicked **Save Changes** in Step 2.
